使用说明

1. 放文件
把本页命名为 index.html，和 4 个 JSON 放进同一个文件夹。
2. 默认文件名
vocab1.json 到 vocab4.json；若你的名字不同，改下面 JS 配置即可。
3. JSON 格式
支持数组、data/words/items/entries、以及按 units 嵌套的常见结构。
可兼容的 JSON 示例
[ {"unit": 1, "word": "hola", "meaning": "你好", "pos": "interj."}, {"unit": 1, "word": "adiós", "meaning": "再见"} ] 或 { "units": [ { "unit": 1, "words": [ {"spanish": "hola", "chinese": "你好"}, {"spanish": "gracias", "chinese": "谢谢"} ] } ] }
